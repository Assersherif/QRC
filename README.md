# QRC — Quantum Reservoir Computing resources

A collection of reference material on **Quantum Reservoir Computing (QRC)** and related
quantum machine learning applications, kept for future reference.

## Contents

### `QORC/`
Reproduced implementation of a **Quantum Optical Reservoir Computing (QORC)** paper,
copied from the MerlinQuantum `reproduced_papers` project.

- **Source:** https://github.com/merlinquantum/reproduced_papers/tree/main/papers/QORC
- **Topic:** quantum reservoir computing applications
- Includes the reproduction notebook (`qorc.ipynb`), supporting library code (`lib/`),
  experiment configs (`configs/`), tests, and result artifacts.

See `QORC/README.md` for the original project documentation and run instructions.

Quandela Paper about QRC: https://arxiv.org/pdf/2605.10471
